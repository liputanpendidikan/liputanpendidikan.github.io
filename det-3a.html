39. apakah det (A+B) = det (A) + det (B)?


Misalkan matriks A = [tex]\left[\begin{array}{ccc}a&b\\c&d\end{array}\right][/tex] dan matriks B = [tex]\left[\begin{array}{ccc}e&f\\g&h\end{array}\right][/tex]
Tentukan dahulu matriks baru (A + B).
=> (A + B) = [tex]\left[\begin{array}{ccc}a&b\\c&d\end{array}\right]+\left[\begin{array}{ccc}e&f\\g&h\end{array}\right][/tex]
=> [tex]\left[\begin{array}{ccc}a+e&b+f\\c+g&d+h\end{array}\right][/tex]
Sekarang, tentukan matriksnya:
=> [tex]\left|\begin{array}{ccc}a+e&b+f\\c+g&d+h\end{array}\right|[/tex]
=> [tex](a+e)(d+h)-(c+g)(b+f)[/tex]
=> [tex]ad+ah+ed+eh-bc-cf-bg-fg[/tex]
Sekarang, tentukan determinan matriks masing masing dan jumlahkan.
=> [tex]\left|\begin{array}{ccc}a&b\\c&d\end{array}\right|[/tex] + [tex]\left|\begin{array}{ccc}e&f\\g&h\end{array}\right|[/tex]
=> [tex](ad-bc)+(eh-gf)[/tex]
=> ad - bc + eh - gf
Dari kedua jawaban diatas, maka det(A+B) ≠ det(A) + det(B)
