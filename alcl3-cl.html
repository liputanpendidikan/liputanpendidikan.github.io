4. Bilangan oksidasi tertinggi dari atom Cl terdapat pada senyawa…A. KCl B. KClO C. CaCl2D. KClO3 E. AlCl3​


Jawaban:

Bilangan oksidasi Cl tertinggi terdapat pada senyawa KClO₃. Dimana dalam senyawa KClO₃ bilangan oksidasi Cl adalah +5.

Penjelasan:

Bilangan oksidasi Cl dalam KCl adalah -1.

KCl

1 + biloks Cl = 0

biloks Cl = -1

Bilangan oksidasi Cl dalam KClO adalah +1.

KClO

1 + biloks Cl + (-2) = 0

biloks Cl = -1 + 2

Biloks Cl = +1

Bilangan oksidasi Cl dalam CaCl₂ adalah -1.

CaCl₂

2 + 2 biloks Cl = 0

2 biloks Cl = -2

biloks Cl = -1

Bilangan oksidasi Cl dalam KClO₃ adalah +6.

KClO₃

1 +  biloks Cl + (-6)= 0

biloks Cl = -1 + 6

biloks Cl = +5

Bilangan oksidasi Cl dalam AlCl₃ adalah -1.

Diurai menjadi Al⁺³ + 3Cl⁻ Sehingga biloks Cl adalah -1

5. 2Al(s) + 3HCl(aq) AlCl3(aq) + 3H2 Jika HCl sebanyak 21 gram telah bereaksi, maka masa AlCl3 yang terbentuk adalah :….(ArAl= 27, Cl= 35,5, H=1)


Logam Al bereaksi dengan HCl membentuk senyawa AlCl3 dan gas Hidrogen sesuai persamaan berikut:

2Al + 6HCl ⇒ 2AlCl3 + 3H2

Jika reaksi menghasilkan 30 gram H2 maka jumlah mol H2 tersebut adalah:

n H2 = gr/Mr = 30/2 = 15 mol

Karena H2 : Al = 3 : 2 maka jumlah Al yang dibutuhkan adalah:

n Al = 2/3 x 15 = 10 mol

gr Al = n x Mr = 10 x 27 = 270 gram

24. Sebanyak 2,7 gram Al bereaksi menurut persamaan reaksi berikut: Al + HCL -> AlCl3 + H2 (Belum Setara) Hitunglah; a. Massa HCL yang diperlukan b. Jumlah molekul AlCl3 yang dihasilkan c. Volume gas H2 Yang diukur pada; -RTP -STP (Ar FH =27, H = 1, Cl = 35,5


(Reaksi setara)

2Al + 6HCl ➡️ 2AlCl3 + 3H2

A :

-> Mol Al = massa Al/ Ar Al = 2,7/27 = 0,1 mol.

-> Maka mol HCl = (6/2) x 0,1 mol = 0,3 mol.

-> Sehingga massa HCl = mol x Mr HCl = 0,3 mol x 36,5 = 10,95 gram

B :

-> Mol AlCl3 = mol Al = 0,1 mol

-> Jumlah partikel AlCl3= mol x 6,02.10^23 = 0,1 x 6,02.10^23 = 0,602x10^23 molekul AlCl3

C :

-> Mol H2 = (3/2) x 0,1 mol = 0,15 mol

-> Volume gas H2 (STP) = mol x 22,4 = 0,15 mol x 22,4 = 3,36 L

-> Volume gas H2 (RTP) = mol x 2,24 = 0,15 mol x 2,24 = 0,336 L

[tex]MgCl_2 \rightarrow Mg^{2+} + 2 Cl^- \\ ~~~~~3~~~~~~~~~~~3~~~~~~~~~6~mmol[/tex]

mol AlCl₃ = M x V = 0,4 x 45 = 18 mmol

[tex]AlCl_3 \rightarrow Al^{3+} + 3Cl^- \\ ~~~18~~~~~~~~18~~~~~~~~~54~mmol[/tex]


[tex]M= \frac{mmol~Cl^-~total}{V~total} \\ \\ = \frac{6+54}{15+45} \\ \\ = \frac{60}{60}=1~M [/tex]

Mol adalah satuan jumlah zat yang digunakan dalam perhitungan kimia. Mol dapat digunakan untuk menyatakan jumlah partikel, massa dan volume suatu zat.  

n = [tex]\frac{massa}{mr}[/tex]

Keterangan:  

     n        = mol suatu zat (mol)  

     massa = massa suatu zat (g)  

     Mr     = massa molekul relatif suatu zat (g/mol)

40. Dari 100gram mineral batuan mengandung 30,4 gram Al2O3 dan AlCl3. Tentukan kadar alumunium dalam batuan tersebut. (Ar Al=27 O=16 Cl=35,5)


Kelas: X
Mata Pelajaran: Kimia
Materi: Stoikiometri
Kata Kunci: Alumunium


Jawaban pendek:

Dari 100 gram mineral batuan mengandung 30,4 gram Al₂O₃ dan AlCl₃.

 

Maka kadar alumunium dalam batuan tersebut adalah 10,4576 %

 

(Ar Al = 27, O = 16, Cl = 35,5)

Jawaban panjang:

Tahap 1: Mencari massa relatif (Mr) Al₂O₃ dan AlCl₃.

 

Massa relatif (Mr) adalah penjumlahan dari massa atom (Ar) penyusunnya.

 

Massa relatif (Mr) Al₂O₃ = 2 Ar Al + 3 Ar  O

                                         = 2 (27) + 3 (16)

                                         = 102

 

Massa relatif (Mr) AlCl₃ = Ar Al + 3 Ar  Cl

                                         = 27 + 3 (35,5)

                                         = 133,5

 

Tahap 2: Mencari presentasi aluminium dalam Al₂O₃ dan AlCl₃.

 

Presentase atau kadar aluminium dalam Al₂O₃ adalah total massa atom aluminium di kedua senyawa dibagi dengan massa relatif (Mr) kedua senyawa.

 

% Al = 3 Ar Al / Mr Al₂O₃ + Mr AlCl₃ x 100%

         = 3 (27)/ (102 + 133,5) x 100%

         =  34,4 %

 

Tahap 3: Mencari kadar Al₂O₃ dan AlCl₃ dalam mineral

 

Presentase atau kadar Al₂O₃ dan AlCl₃ adalah total massa kedua senyawa di mineral dibagi dengan massa mineral.

 

% Al₂O₃ dan AlCl₃ = m Al₂O₃ dan AlCl₃ / m Mineral x 100%

                               = (30,4 gr/ 100 gr) x 100%

                              =  30,4 %

 

 

 

Tahap 4: Mencari kadar aluminium dalam mineral

 

Kadar aluminium dalam mineral adalah perkalian dari kadar Al₂O₃ dan AlCl₃ dalam mineral dan kadar aluminium dalam Al₂O₃ dan AlCl₃, dibagi 100%.

 

Kadar Al = %Al x % Al₂O₃ dan AlCl₃ / 100%

                = 34,4 % x 30,4 % / 100%

                = 10,4576 %
