3. penjumlahan 1 + 3 + 5 + 7 + ... + 55 = ...​


Kita juga tulis sebagai

[tex] 2(0)+1 + 2(1)+1 + 2(2)+1 + 2(3)+1 + \dots + 2(27)+1 = \sum_{k = 0}^{27} [2k+1] \\ [/tex]

Untuk ruas kanan, kita dapat tulis sebagai

[tex] \sum_{k = 1}^{28} [2(k - 1) + 1] = \sum_{k = 1}^{28} [2k - 1] \\[/tex]

Kita dapat tulis sebagai

[tex] 2 \sum_{k = 1}^{28} k - \sum_{k = 1}^{28} 1 = 2 \cdot \frac{28 \cdot 29}{2} - 28 = 28(29 - 1) = 28^2 \\ [/tex]

Oleh karena itu,

[tex] 1 + 3 + 5 + 7 + \dots + 55 = 28^2 = 784 [/tex]

10. Misalkan m=1+3+5+7+9+11+.....+2023 sisa dari m dibagi 55 adalah


Untuk menghitung sisa dari m dibagi 55, kita bisa menggunakan aturan deret aritmetika.

Deret ini adalah deret bilangan ganjil berturut-turut, dimulai dari 1 hingga 2023. Untuk menentukan berapa banyak suku ganjil yang ada dalam deret ini, kita bisa menggunakan rumus berikut:

S_n = (n/2) * [2a + (n-1)d]

Di mana:
- S_n adalah jumlah n suku pertama dalam deret.
- n adalah banyaknya suku.
- a adalah suku pertama (1).
- d adalah selisih antara suku-suku (2, karena ini adalah deret bilangan ganjil).

Kita ingin mencari n sehingga S_n ≤ 2023, tetapi S_(n+1) > 2023. Kita akan mencoba nilai-nilai n sampai kita menemukan nilai yang memenuhi kondisi tersebut.

Misalnya, untuk n = 63:

S_63 = (63/2) * [2*1 + (63-1)*2]
S_63 = 31.5 * [2 + 124]
S_63 = 31.5 * 252 = 7938

S_64 = (64/2) * [2*1 + (64-1)*2]
S_64 = 32 * [2 + 126]
S_64 = 32 * 256 = 8192

Dengan demikian, jumlah 63 suku pertama adalah 7938, yang kurang dari 2023, dan jumlah 64 suku pertama adalah 8192, yang lebih dari 2023. Jadi, kita akan menggunakan 63 suku pertama dalam perhitungan kita.

Sekarang, kita dapat menghitung sisa dari m dibagi 55:

m = 1 + 3 + 5 + ... + 2023
m = S_63

Sekarang, kita hitung m dibagi 55:

m mod 55 = 7938 mod 55

Kita dapat menghitungnya sebagai berikut:

7938 = 55 * 144 + 18

Jadi, sisa dari m dibagi 55 adalah 18.

32. 1+3+5+7...+55=a.780b.782c.781d.784​


Jumlah barisan aritmatika : 1 + 3 + 5 + 7 + ... + 55 adalah 784.

Penyelesaian Soal :

Diketahui : Barisan = 1 + 3 + 5 + 7 + ... + 55

                  Suku pertama (a) = 1

                  Suku terakhir (Un) = 55

Ditanya : Jumlah barisan (Sn) ?

Jawab :

LANGKAH PERTAMA (I)

Hitung beda (b) barisan dengan menggunakan cara sebagai berikut :

b = U₂ - U₁

  = 3 - 1

  = 2

LANGKAH KEDUA (II)

Hitung banyaknya suku (n) dengan menggunakan rumus sebagai berikut :

Uₙ = a + (n - 1) b

Keterangan : Uₙ = suku ke-n

                      a = suku pertama

                      b = beda

                      n = banyaknya suku

Maka cara menghitungnya yaitu :

Uₙ = a + (n - 1) b

55 = 1 + (n - 1) 2

55 = 1 + 2n - 2

55 = 2n - 1

2n = 55 + 1

2n = 56

n = 56/ 2

n = 28

LANGKAH KETIGA (III)

Hitung jumlah barisan (Sn) dengan menggunakan rumus sebagai berikut :

Sₙ = n/2 (a + Uₙ)

Keterangan : Sₙ = Jumlah suku n pertama

                      Uₙ = suku ke-n

                      a = suku pertama

                      n = banyaknya suku

Maka cara menghitungnya yaitu :

Sₙ = n/2 (a + Uₙ)

   = 28/2 (1 + 55)

   = 14 (56)

   = 784

∴ Kesimpulan nilai jumlah barisan (Sn) artimatika tersebut adalah 784.
