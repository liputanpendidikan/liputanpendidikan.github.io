7. Mohon bantuannya untuk soal TPS​


Misal, bilangan pada kolom pertama = A dan bilangan pada kolom kedua = B, maka pola operasi aritmetiknya adalah:

A*(A - 1) - B

Sehingga, untuk baris pertama dan ketiga akan didapatkan:

5*(5 - 1) - 3 = 5*4 - 3 = 20 - 3 = 7

7*(7 - 1) - 2 = 7*6 - 2 = 42 - 2 = 40

Sehingga, untuk baris kedua, akan didapatkan:

6*(6 - 1) - x = 25

6*5 - x = 25

30 - x = 25

x = 30 - 25 = 5

26. contoh soal sbmptn tps barisan dan deret


Contoh soal

Diketahui a, b, dan c berturut-turut adalah suku ke-2, ke-3 dan ke-4 suatu barisan geometri dengan b>0. Jika [tex]\frac{ac}{b+2}=1[/tex] maka nilai b adalah...

Penyelesaian

Diketahui a, b, dan c rturut-turut adalah suku ke-2, ke-3 dan ke-4

maka dalam penulisan barisan dan deret dapat ditulis

x, a, b, c

Karena barisan merupakan barisan geometri, maka dalam mencari rasio (r), dicari dengan membagi suatu suku dengan suku sebelumnya. [tex]\frac{Suku ke-3}{Suku ke-2} =r = \frac{b}{a}\\\\ \frac{Suku ke-4}{Suku ke-3} =r = \frac{c}{b}[/tex]

Karena rasio bernilai sama, maka didapatkan

[tex]\frac{b}{a} =\frac{c}{b}\\\\ b^{2}=ac \\atau \\ac = b^{2}[/tex]

Diketahui pada soal  [tex]\frac{ac}{b+2}=1[/tex]

[tex]ac=b+2\\[/tex]

substitusi nilai [tex]ac = b^{2} \\[/tex]

[tex]b^{2}=b+2\\b^{2}-b-2=0\\(b-2)(b+1)=0[/tex]

b=2 atau b=-1

Karena nilai rasio atau pembeda tidak bernilai negatif

Maka  nilai b  adalah 2.

29. tolong bantu kk, . latihan soal tiu cpns


Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)

38. tolong dibantu soal tps ni kawan2 ​
