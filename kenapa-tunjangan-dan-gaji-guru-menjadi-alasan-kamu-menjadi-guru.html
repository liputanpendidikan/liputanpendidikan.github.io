6. Apa yang dipersiapkan guru dan sekolah menuju kurikulum merdeka dari segi kompetensi guru,mindset guru dan infrastruktur penunjang


Analisis Materi Esensial

Salah satu hal yang menarik dari kurikulum 2022 yakni materi pembelajaran yang esensial, hanya pada hal-hal yang dibutuhkan siswa saja, sehingga guru perlu menganalisis materi esensial. Penyediaan waktu yang memadai untuk kompetensi penunjang keterlaksanaan program wajib seperti pembelajaran berbasis proyek untuk semua jenjang. Perhitungan waktu-waktu wajib di kegiatan pembelajaran harus tersedia. disesuaikan dengan karakteristik sekolah dan siswa.

Dalam menganalisis Materi Esensial perlu memperhatikan capaian pembelajaran yang kurikulum 2022 atau yang sudah diterapkan di sekolah penggerak mengacu pada Keputusan Balitbangbu No. 028/H/KU/2021 . Dan memperhatikan struktur kurikulum serta alokasi waktu pembelajaran sesuai Keputusan Mendikbud No. 371/M/2021 tentang Program Sekolah Penggerak.

perform Kondisi dan Keterlibatan Siswa

Selama pembelajaran keberanian, guru patut mempertimbangkan kondisi dan keterlibatan siswa. karena Kondisi selama pembelajaran berani merupakan penyebab utama hilangnya pembelajaran. Pembelajaran pada PTM terbatas banyak menemukan perbedaan pemahaman tentang penguasaan materi karena berbagai faktor. Pembelajaran yang patut disiapkan sekolah dalam mengatasi masalah ini. Pada PTM terbatas secara ketat diharapkan perlahan-lahan dapat menjadi pembiasaan yang selama ini tidak terlaksana selama pembelajaran berani. Jangan sampai siswa kehilangan lagi kesempatan untuk belajar melalui pengalaman langsung.

22. Diketahui gaji p.surya semula Rp 1.800.000, dan tunjangan jabatan Rp 200.000, dikonsumsi sebanyak Rp 1.600.000 perbulan, kemudian gajinya naik menjadi Rp 2.000.000 dan tunjangan jabatan Rp 300.000, sedangkan kenaikan konsumsi bertambah menjadi Rp 1.850.000 perbulan, dari data di atas carilah besarnya mpc dan mps


Jawaban:

Penjelasan:

MPC (Marginal Propensity to Consume) adalah proporsi kenaikan konsumsi terhadap kenaikan pendapatan. MPS (Marginal Propensity to Save) adalah proporsi kenaikan penghematan terhadap kenaikan pendapatan.

Untuk menghitung MPC:

MPC = (Kenaikan konsumsi / Kenaikan pendapatan) x 100%

MPC = (850.000 - 1.600.000) / (2.000.000 - 1.800.000) x 100% = -0.2 x 100% = -20%

Maka dapat diketahui bahwa MPC dari P.Surya sebesar -20%. Ini menunjukkan bahwa untuk setiap kenaikan pendapatan sebesar Rp 200.000, konsumsinya akan turun sebesar Rp 200.000 * (-20%) = Rp 40.000.

Untuk menghitung MPS:

MPS = (Kenaikan penghematan / Kenaikan pendapatan) x 100%

MPS = (300.000 - 200.000) / (2.000.000 - 1.800.000) x 100% = 100.000 / 200.000 x 100% = 50%

Maka dapat diketahui bahwa MPS dari P.Surya sebesar 50%. Ini menunjukkan bahwa untuk setiap kenaikan pendapatan sebesar Rp 200.000, penghematannya akan meningkat sebesar Rp 200.000 * (50%) = Rp 100.000.

29. Buatlah program untuk menghitung gaji pegawai Input : nama, golongan Proses : · Jika golongan A maka gaji pokok =Rp 1.500.000, tunjangan =5% * gaji pokok .jika gol.B maka gaji pokok = Rp.1.700.000,tunjangan = 10%* gaji pokok .jika gol.C maka gaji pokok = Rp.2.000.000, tunjangan = 15%*gaji pokok Output : Nama,gaji pokok,tunjangan serta gaji bersih Petunjuk : gunakan array dan petualangan


program untuk menghitung gaji pegawai, program ditulis dalam bahasa C

#include <stdio.h>

int main()

{

    char nama[50];

    char golongan;

    int gaji[3] = { 1500000, 1700000, 2000000 };

    int persentunjangan[3] = { 5, 10, 15 };

    int gajibersih, tunjangan;

    printf("Masukan nama Anda : ");

    scanf("%s", nama);

    printf("Masukan golongan (A, B, atau C) : ");

    scanf("%c", &golongan);

    golongan = golongan - 'A';

    tunjangan = gaji[ golongan ] * persentunjangan[ golongan ] / 100;

    gajibersih = gaji[ golongan ] + tunjangan;

    printf("Nama : %s\n", nama);

    printf("Gaji pokok : %i\n", gaji[ golongan ]);

    printf("Tunjangan : %i\n", tunjangan);

    printf("Gaji bersih : %i\n", gajibersih);

    return 0;

}

Pembahasan

Ada bagian yang aneh pada soal, "gunakan array dan petualangan". Array adalah bentuk variabel dengan alokasi memory untuk banyak nilai. Yang saya bingung adalah istilah "petualangan" disini. Setahu saya tidak ada istilah "petualangan" dalam bahasa pemrograman, mohon dikoreksi jika memang terjadi typo.

Pada solusi pemrograman yang dibuat, kita memanfaatkan golongan untuk menjadi index dari Array.

Pada potongan code berikut golongan = golongan - 'A';

Berfungsi untuk mengubah karakter A menjadi nilai 0. Perlu diketahui karakter adalah sebuah huruf yang memiliki nilai angka pada komputer, atau ASCII. Sehingga bisa dioperasikan menggunkana operator aritmatika seperti penjumlahan pengurangan, dan lainnya.

Maka jika golongan adalah A dan dikurangi 'A' maka hasilnya adalah 0

jika golongan adalah B dan dikurangi 'A' maka hasilnya adalah 1

jika golongan adalah C dan dikurangi 'A' maka hasilnya adalah 2

Nilai ini bisa kita gunakan sebagai index dalam array gaji yang sudah didefinisikan sebelumnya.

Sehingga kita bisa akses secara dinamis array gaji menggunakan golongan sebagi indexnya.
