40. himpunan penyelesaian dari sin 3x + sin x - sin 2x = 0


Trigonometri terdiri dari sinus, kosinus, tangen, secan, kosecan, kotangen. Perbandingan trigonometri hanya berlaku pada segitiga siku-siku. Untuk segitiga sembarang, bisa menggunakan aturan sinus dan aturan kosinus

Rumus penjumlahan dan pengurangan trigonometri

sin A + sin B = 2 sin ½ (A + B) cos ½ (A – B) sin A  –  sin B = 2 cos ½ (A + B) sin ½ (A – B) cos A + cos B = 2 cos ½ (A + B) cos ½ (A – B) cos A  –  cos B =  – 2 sin ½ (A + B) sin ½ (A – B)

Persamaan Trigonometri

1) sin x = sin α

x = α + k . 360° atau x = (180° – α) + k . 360°

2) cos x = cos α

x = α + k . 360 atau x = –α + k . 360°

3) tan x = tan α

x = α + k . 180°


Pembahasan

sin 3x + sin x – sin 2x = 0

2 sin ½ (3x + x) cos ½ (3x – x) – sin 2x = 0

2 sin ½ (4x) cos ½ (2x) – sin 2x = 0

2 sin 2x cos x – sin 2x = 0

sin 2x (2 cos x – 1) = 0

sin 2x = 0 atau (2 cos x – 1) = 0


Misal interval dari x adalah 0° ≤ x ≤ 360°


1) sin 2x = 0

   sin 2x = sin 0°

         2x = 0° + k . 360° atau 2x = (180° – 0°) + k . 360°

           x = k . 180°                    x = 90° + k . 180°

k = 0 => x = 0°                           x = 90°

k = 1 => x = 180°                        x = 270°

k = 2 => x = 360°                     x = 450° (TM)

k = 3 => Tidak ada nilai x yang memenuhi


2) (2 cos x – 1) = 0

    2 cos x = 1

       cos x = ½  

       cos x = cos 60°

            x = 60° + k . 360° atau x = –60° + k . 360°

k = 0 => x = 60°                          x = –60° (TM)

k = 1 => x = 420° (TM)                x = 300°  

k = 2 => Tidak ada nilai x yang memenuhi


Jadi himpunan penyelesaiannya adalah

HP = {0°, 60°, 90°, 180°, 270°, 300°, 360°}


Jika kita ubah menjadi dalam bentuk π radian

0° = [tex]\frac{0}{180}\pi = 0[/tex]  

60° = [tex]\frac{60}{180}\pi = \frac{1}{3}\pi[/tex]

90° = [tex]\frac{900}{180}\pi = \frac{1}{2}\pi[/tex]

180° = [tex]\frac{180}{180}\pi = \pi[/tex]

270° = [tex]\frac{270}{180}\pi = \frac{3}{2}\pi[/tex]

300° = [tex]\frac{300}{180}\pi = \frac{5}{3}\pi[/tex]

360° = [tex]\frac{360}{180}\pi = 2\pi[/tex]

Jadi himpunan penyelesaiannya adalah

HP = {[tex]0, \: \frac{1}{3}\pi, \: \frac{1}{2}\pi, \: \pi, \: \frac{3}{2}\pi, \: \frac{5}{3}\pi, \: 2\pi[/tex]}
