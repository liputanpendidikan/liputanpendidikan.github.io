7. sin2 A + cos2 A = .....​


Jawab:

Penjelasan dengan langkah-langkah:

segitiga siku siku

ΔABC siku di B

i) pythagoras  

AB²+ BC² = AC²

ii) sin A = BC/AC

cos A = AB/AC

iii) sin² A  + cos² A =  BC²/AC²  + AB²/AC²

sin² A  + cos² A =  (BC² + AB²)/AC²

sin² A  + cos² A = AC² /AC²

sin² A  + cos² A = 1

Penjelasan dengan langkah-langkah:

[tex] \sin A = \frac{y}{r} [/tex]

[tex] \cos A = \frac{x}{r} [/tex]

maka :

[tex] \sin^{2} A + \cos^{2} A [/tex]

[tex] = \frac{ {y}^{2} }{ {r}^{2} } + \frac{ {x}^{2} }{ {r}^{2} } [/tex]

[tex] = \frac{( {x}^{2} + {y}^{2}) }{ {r}^{2} } [/tex]

Menggunakan konsep teorema pythagoras :

[tex]{r}^{2} = {x}^{2} + {y}^{2} [/tex]

maka :

[tex] = \frac{ {r}^{2} }{ {r}^{2} } [/tex]

[tex] = 1[/tex]

Jadi, nilai sin² A + cos² A = 1

28. Turunan dari f(x) = sin x cos x adalaha. -cos 2xb. cos 2xC. cos2 xd. cos2 x + sin2 xe. -cos2 x - sina x​


Turunan dari f(x) = sinx cosx adalah f'(x) = cos 2x

Pembahasan

Turunan fungsi trigonometri adalah turunan yang memuat fungsi trigonometri seperti sinus, cosinus, tangen, cosecan, secan, dan cotangen.

Turunan dasar fungsi trigonometri :

f(x) = sin x maka f'(x) = cos x

f(x) = cos x maka f'(x) = - sin x

f(x) = tan x maka f'(x) = sec²x

f(x) = sec x maka f'(x) = sec x tan x

f(x) = cosec x maka f'(x) = - cosec x cotan x

f(x) = cotan x maka f'(x) = - cosec²x

Operasi pada turunan

1. penjumlahan dan pengurangan

f(x) = u ± v maka f(x) = u' ± v'

2. perkalian

f(x) = u x v maka f'(x) = u' v + u v'.

Beberapa rumus sudut rangkap

1. cos 2x = 1 - 2 sin² x atau

cos 2x = 2 cos² x - 1 atau

cos 2x = cos² x - sin² x

2. sin 2x = 2 sinx cosx

===========================================

Diketahui :

[tex]f(x) = sinx \: cosx[/tex]

Ditanya :

[tex]f'(x) = ....[/tex]

Jawab :

[tex]f(x) = sinx \: cosx[/tex]

fungsi di atas merupakan bentuk perkalian dua fungsi, yaitu

u = sinx dan v = cos x

maka

u' = cos x dan v' = - sinx

Sehingga, turunan pertama dari f(x) adalah

[tex]f'(x) = u' \: v \: + \: u \: v'[/tex]

[tex]f'(x) =cos \: x( cos \: x) + sin \: x ( - sin \: x) [/tex]

[tex]f'(x) =cos{}^{2} x - sin {}^{2} \: x [/tex]

dimana, cos²x - sin²x = cos2x, maka,

[tex]f'(x) =cos \: 2 x [/tex]

Jadi, turunan pertama dari f(x) = sinx cos x adalah f'(x) = cos 2x

32. buktikan bahwa sin2 k+cos2 k =1


ubah ke kartesius.

x = r.sin(k)
sin(k) = x/r

y = r.cos(k)
cos(k) = y/r

sin^2(k) + cos^2(k) = (x/r)^2 + (y/r)^2
= (x^2 + y^2)/r^2

Karena r^2 = x^2 + y^2 (berdasarkan pitagoras), maka :

= r^2/r^2
= 1

==========

Cara lainnya.

f(k) = sin^2(k) + cos^2(k)
f'(k) = 2.sin(k).cos(k) - 2.cos(k).cos(k)
f'(k) = 0

Karena f'(k) = 0, maka f(k) = c, c suatu konstanta.

Misal, k = 0, maka :

sin^2(k) + cos^2(k) = sin^2(0) + cos^2(0) = 0 + 1 = 1
