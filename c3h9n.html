8. .Suatu gas hidrokarbon tersusun atas 80% atom karbon dan sisanya hydrogen. Dalam 3 gram gas tersebut terdapat 6,02 x 10^22 molekul. Jika Ar C= 12, H=1 maka rumus molekul hidrokarbon tersebut adalah • CH2 CH3 • C2H4 • C2H6 • C3H9


Jawaban:

C2H6

Penjelasan:

Jumlah molekul = n x 6,02x10^23

6,02x10^22 = n x 6,02x10^23

n = 0,1 mol

massa CxHy = n x Mr

3 = 0,1 x Mr

Mr = 30

80℅ atom C x 30 = 24

Ar C = 12 berarti jumlah atom C = 24 : 12 = 2

Mr CxHy = 30

( 2 x Ar C ) + ( y x Ar H ) = 30

24 + ( y x Ar H ) = 30

maka nilai y = 6

sehingga rumus kimia CxHy : C2H6

11. 2. Setarakan reaksi kimia berikut ini a. CH4 +O₂→CO₂ + H₂O b. C3H6 +NH3 + O₂ → C3H3N +3H₂O C. NH3→N₂ + H₂​


a. CH4 + O2 → CO2 + H2O

Untuk menyeimbangkan reaksi ini, perhatikan jumlah atom di kedua sisi persamaan:

- Atom karbon (C): 1 atom di sebelah kiri, 1 atom di sebelah kanan

- Atom hidrogen (H): 4 atom di sebelah kiri, 4 atom di sebelah kanan

- Atom oksigen (O): 2 atom di sebelah kiri (dari O2), 2 atom di sebelah kanan (dari CO2 dan H2O)

Dengan demikian, reaksi tersebut sudah seimbang secara atomik.

b. C3H6 + NH3 + O2 → C3H3N + 3H2O

Perhatikan jumlah atom di kedua sisi persamaan:

- Atom karbon (C): 3 atom di sebelah kiri, 3 atom di sebelah kanan

- Atom hidrogen (H): 9 atom di sebelah kiri, 9 atom di sebelah kanan

- Atom nitrogen (N): 1 atom di sebelah kiri, 1 atom di sebelah kanan

- Atom oksigen (O): 2 atom di sebelah kiri (dari O2), 2 atom di sebelah kanan (dari C3H3N), ditambah 3 atom di sebelah kanan (dari 3H2O)

Reaksi ini belum seimbang dalam hal atom hidrogen dan atom oksigen. Untuk menyeimbangkan, kita dapat menambahkan koefisien sebagai berikut:

C3H6 + NH3 + 7/2 O2 → C3H3N + 3H2O

Dengan ini, jumlah atom hidrogen dan oksigen di kedua sisi persamaan menjadi seimbang.

c. NH3 → N2 + H2

Perhatikan jumlah atom di kedua sisi persamaan:

- Atom nitrogen (N): 1 atom di sebelah kiri, 2 atom di sebelah kanan

- Atom hidrogen (H): 3 atom di sebelah kiri, 2 atom di sebelah kanan

Reaksi ini belum seimbang dalam hal atom nitrogen. Untuk menyeimbangkan, kita dapat menambahkan koefisien sebagai berikut:

2NH3 → N2 + 3H2

Dengan ini, jumlah atom nitrogen di kedua sisi persamaan menjadi seimbang.

CH4 + 2O₂ → CO₂ + 2H₂O

b. Untuk setarakan reaksi kimia C3H6 + NH3 + O₂ → C3H3N + 3H₂O, kita perlu mengimbangi jumlah atom karbon (C), hidrogen (H), dan nitrogen (N) di kedua sisi persamaan. Jadi, persamaan yang setara adalah:

C3H6 + 3NH3 + 3O₂ → C3H3N + 6H₂O

c. Untuk setarakan reaksi kimia NH3 → N₂ + H₂, kita perlu mengimbangi jumlah atom nitrogen (N) dan hidrogen (H) di kedua sisi persamaan. Jadi, persamaan yang setara adalah:

2NH3 → N₂ + 3H₂

Dengan mengimbangi reaksi kimia, kita memastikan bahwa jumlah atom dari setiap unsur di kedua sisi reaksi sama.
