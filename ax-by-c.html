Pengertian

sistem persamaan linear satu variabel ialah suatu persamaan yang memiliki variabel dimana jumlah variabelnya hanya satu .

Pembahasan dan penjelasan

salah bentuk umum dari persamaan linear satu variabel yaitu

[tex] \boxed{ax + b \: = c}[/tex]

adapun contoh-contoh soal dari persamaan linear satu variabel

2x + 4 = 6x - 3 = 5

Contoh soal

2x - 4 = 6

2x = 6 + 4

2x = 10

x = 10/2

x = 5

Persamaan linear terbagi menjadi 3

persamaan linear satu variabelpersamaan linear dua variabelpersamaan linear tiga variabel

╭┈─────── ೄྀ࿐ ˊˎ╰┈─➤ Pembahasan Materi ☽

Persamaan kuadrat merupakan salah satu persamaan yang dimana pangkat tertinggi dari variabel pembentuk umumnya adalah dua. Persamaan kuadrat ini terdiri dari empat jenis yaitu :

Murni => nilai b = 0Biasa => nilai a = 1Tidak lengkap => nilai c = 0Rasional => terdiri dari bilangan rasional.

Variabel yang diberikan adalah x dan y. Sehingga bentuk umumnya adalah : ax² + bx + c = 0 . Maka, jawaban yang benar adalah c.

29. menentukan garis singgung lingkaran X2 + Y2- ax- by+ c =0 yang tegak lurus dengan garis ax- y+ c=0


Dengan gradien garis ax - y + c adalah a
Dan gradien tegak lurusnya adalah -1/a

Dengan persamaan lingkaran:
0 = x² + y² - ax - by + c
0 = (x² - ax) + (y² - by) + c
0 = (x² - ax + 1/4 a²) + (y² - by + 1/4 b²) + c - 1/4 (a²+b²)
(x - 1/2 a)² + (y- 1/2 b)² = 1/4 (a²+b²) - c

Dengan pusat (a/2, b/2), dan jari-jari 1/2 √a²+b²-4c

Persamaan garis singgung:
y - yp = m(x - xp) +- r√[1+m²]
y - b/2 = -1/a (x - a/2) +- 1/2 √[(a²+b²-4c)(1+(1/a)²)]
y - b/2 = -x/a + 1/2 +- 1/2 √[(a²+b²-4c) 1/a² (a²+1)]
y = -1/a x + 1/2 (b+1) +- 1/(2a) √[(a²+b²-4c)(a²+1)]
