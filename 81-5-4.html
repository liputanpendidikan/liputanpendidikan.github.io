5. 3. Taksiran dari adalah.... 81/5 - 4/5 adalah​


Untuk menyelesaikan masalah ini, pertama-tama kita harus menyederhanakan kedua sisi persamaan. Pada sisi kiri, kita dapat menyederhanakan 81/5 - 4/5 menjadi 81/5 - 16/5. Sementara itu, pada sisi kanan, kita dapat menyederhanakan adalah menjadi adalah. Setelah menyederhanakan kedua sisi persamaan, kita dapat menuliskannya sebagai berikut:

81/5 - 16/5 = adalah

Selanjutnya, kita dapat mengurangkan kedua sisi persamaan untuk mencari nilai adalah. Ini akan menghasilkan:

81/5 - 16/5 - 16/5 = adalah - 16/5

65/5 = adalah - 16/5

13 = adalah - 16/5

Kemudian, kita dapat menambahkan 16/5 ke kedua sisi persamaan untuk mencari nilai adalah:

13 + 16/5 = adalah - 16/5 + 16/5

33/5 = adalah

Jadi, taksiran dari adalah adalah 33/5.

7. nilai dari 1. 16^-1/42.23^-1/53. 81^ 1/44. 125^ -2/3 5. 81^ 3/4 ​


Jawab:

Untuk menghitung nilai dari eksponen negatif atau pecahan pada bilangan pangkat, kita dapat menggunakan aturan perpangkatan. ini hasil perhitungan untuk masing-masing ekspresi:

1. 16^(-1/4) = 1 / 16^(1/4) = 1 / √(16) = 1 / 2 = 0.5

2. 23^(-1/5) = 1 / 23^(1/5) ≈ 1 / 1.7411011 ≈ 0.5743492

3. 81^(1/4) = √(81) = √(3^4) = 3

4. 125^(-2/3) = 1 / 125^(2/3) = 1 / (125^(1/3))^2 = 1 / (5^2) = 1 / 25 = 0.04

5. 81^(3/4) = √(81^(3/2)) = √(81^(1/2) * 81^(1/2) * 81) = √(9 * 9 * 81) = √(6561) = 81

Jadi, nilai dari masing-masing ekspresi adalah sebagai berikut:

1. 16^(-1/4) = 0.5

2. 23^(-1/5) ≈ 0.5743492

3. 81^(1/4) = 3

4. 125^(-2/3) = 0.04

5. 81^(3/4) = 81
