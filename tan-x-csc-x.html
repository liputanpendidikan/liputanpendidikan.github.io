identitas trigonometri

sec x = 1/cos x

tan x = sin x / cos x

sin² x + cos² x = 1

•

√((sec x - tan x)/(sec x + tan x))

= √(((1 - sin x)/cos x) / ((1 + sin x)/cos x))

= √((1 - sin x ) / (1 + sin x))

= √((1 - sin x)(1 - sin x) / (1 + sin x)(1 - sin x))

= √((1 - sin x)² / (1 - sin² x)

= √((1 - sin x)² / cos² x)

= (1 - sin x)/cos x

= 1/cos x - sin x/cos x

= sec x - tan x

5. Buktikan bahwa operasi matematika di bawah nilainya sama dengan tan^2 x [tex]\frac{(csc^{2}x)(sec^2x)-(csc^{2}x) }{(sec^2x)(csc^2x)-(1-tan^2x)} = tan^2x[/tex]


Untuk bagian pembilang, kita faktorkan

[tex] \csc^2 x (\sec^2 x - 1) = \csc^2 x \cdot \tan^2 x [/tex]

dan penyebutnya,

[tex] \sec^2 x \csc^2 x - \sec^2 x = \sec^2 x (\csc^2 x - 1) = \sec^2 x \cdot \cot^2 x [/tex]

Sekarang,

[tex] \frac{\csc^2 x \cdot \tan^2 x}{\sec^2 x \cdot \cot^2 x} = \frac{\frac{1}{\sin^2 x} \tan^2 x}{\frac{1}{\cos^2 x} \cot^2 x} \\ [/tex]

[tex] \frac{\csc^2 x \cdot \tan^2 x}{\sec^2 x \cdot \cot^2 x} = \frac{\cos^2 x}{\sin^2 x} \cdot \frac{\tan^2 x}{\cot^2 x} \\ [/tex]

[tex] \frac{\csc^2 x \tan^2 x}{\sec^2 x \cdot \cot^2 x} = \cot^2 x \cdot \frac{\tan^2 x}{\cot^2 x} = \tan^2 x \, \blacksquare \\ [/tex]

29. tan(270°+x) csc(90°+x):sec(360°−x) cot(90°−x)


Jawab&Penjelasan dengan langkah-langkah:

Mari kita pecahkan dan perhitungkan setiap suku dalam ekspresi tersebut:

1. Tan(270°+x):

  Kita dapat menggunakan siklus periodik trigonometri untuk mencari nilai tan(270°+x). Dalam siklus tersebut, nilai tan(270°) adalah tak terdefinisi (undefined), tetapi kita dapat mencari nilai tan(270°+x) dengan menggunakan siklus sin dan cos.

  Dalam siklus sin dan cos, sin(270°) = -1 dan cos(270°) = 0.

  Jadi, tan(270°+x) = sin(270°+x) / cos(270°+x) = (-1) / 0 = undefined.

2. Csc(90°+x):

  Csc(90°+x) adalah kebalikan dari sin(90°+x). Dalam siklus sin, sin(90°) = 1.

  Jadi, csc(90°+x) = 1 / sin(90°+x).

3. Sec(360°−x):

  Sec(360°−x) adalah kebalikan dari cos(360°−x). Dalam siklus cos, cos(360°) = 1.

  Jadi, sec(360°−x) = 1 / cos(360°−x).

4. Cot(90°−x):

  Cot(90°−x) adalah kebalikan dari tan(90°−x). Dalam siklus sin dan cos, sin(90°) = 1 dan cos(90°) = 0.

  Jadi, cot(90°−x) = cos(90°−x) / sin(90°−x) = 0 / 1 = 0.

Jadi, kita memiliki:

Tan(270°+x) csc(90°+x):sec(360°−x) cot(90°−x) = undefined * (1 / sin(90°+x)) * (1 / cos(360°−x)) * 0 = undefined * 0 = undefined.

Jadi, ekspresi tersebut tidak memiliki nilai yang terdefinisi (undefined).
