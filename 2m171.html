dik:

d = 14 cm

t = 24 cm

dit

Luas Permukaan=...?

jawab:

r = 1/2 x d

= 1/2 x 14 cm

= 7 cm

s = √r²+t²

= √7²+24²

= √49+576

= √625

= 25 cm

luas selimut kerucut

luas selimut = π x r x s

= 22/7 x 7 x 25

= 22 x 25

= 550 cm²

luas alas = π x r x r

= 22/7 x 7 x 7

= 22 x 7

= 154 cm²

luas permukaan

= luas alas + luas selimut

= 154 + 550

= 704 cm²

16. 11. Sebuah bola pejal bermassa 3 kg memiliki jari-jari 5 cm. Bola tersebut kemudian diputar padasumbunya melalui pusat bola dengan kecepatan120 rpm. Momentum sudut bola sebesar ...a. 371 x 10-3 kgm/sb. 871 x 10-3 kgm/s1,210 x 10-2 kgm2/sd. 1,510 10-2 kgm2/se. 2,171 10-2 kgm/s​


DINAMIKA ROTASI

bola pejal

m = 3 kg

R = 5 cm = 5×10⁻² m

f = 120 rpm = 120/60 = 2 Hz

momen inersia bola pejal

I = ⅖ m R²

I = ⅖ • 3 • (5×10⁻²)²

I = 30×10⁻⁴ kg m²

kecepatan sudut

ω = 2π f

ω = 2π • 2

ω = 4π rad/s

momentum sudut

L = I ω

L = 30×10⁻⁴ • 4π

L = 120π×10⁻⁴

L = 0,03768

L = 3,768×10⁻² kg m²/s ✔️
