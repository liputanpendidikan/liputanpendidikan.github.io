5. sebuah limas dengan alasnya berbentuk persegi panjang jika panjang,jika panjang 32cmdan lebar 18cm dan tinggi limas 42cm.Tentukan a)luas alas b)LPL c)Volume limas


Bangun limas adalah bangun ruang yang memiliki alas berbentuk segi-n. Rumus volume nya adalah :

Volume limas = [tex] \frac {1} {3}[/tex] x luas alas x tinggi

Luas alas limas tergantung dengan bentuk alas nya.

=> Alas limas jika berbentuk segitiga :

    Luas alas = [tex] \frac {1} {2}[/tex] x alas segitiga x tinggi segitiga

=> Alas limas jika berbentuk segiempat :

    Luas alas = panjang x lebar

=> Alas limas jika berbentuk trapesium :

    Luas alas = [tex] \frac {1} {2}[/tex] x jumlah sisi sejajar x tinggi trapesium

Sedangkan untuk rumus luas permukaan limas adalah :

Luas permukaan limas = luas alas + jumlah luas sisi-sisi tegak limas.

 

Pembahasan

Perhatikan gambar terlampir

Panjang OP = [tex] \frac {1} {2}[/tex] x panjang AB = [tex] \frac {1} {2}[/tex] x 32 cm = 16 cm

Panjang OS = [tex] \frac {1} {2}[/tex] x panjang BC = [tex] \frac {1} {2}[/tex] x 18 cm = 9 cm

a. Luas alas limas adalah :

= panjang x lebar

= 32 cm x 18 cm  

= 576 cm²

b. Menghitung luas permukaan limas

Hitung panjang PT dan ST menggunakan rumus phytagoras.

Dalil phytagoras : s² = a² + t²

dimana :

s = sisi miring segitiga

a = alas segitiga

t = tinggi segitiga

Menghitung panjang PT, yaitu :

PT² = OT² + OP²

PT² = 42² + 16²

PT² = 1.764 + 256

PT² = 2.020

PT = √2.020

PT = 44,9 cm

dibulatkan menjadi PT = 45 cm

Menghitung panjang ST, yaitu :

ST² = TO² + OS²

ST² = 42² + 9²

ST² = 1.764 + 81

ST² = 1.845

ST = √1.845

ST = 42,9 cm

dibulatkan menjadi ST = 43 cm

Luas sisi tegak limas BCT = luas sisi tegak limas ADT, yaitu :

= [tex] \frac {1} {2}[/tex] x alas x tinggi  

= [tex] \frac {1} {2}[/tex] x 18 cm x 45 cm

= [tex] \frac {1} {2 :2}[/tex] x ( 18 : 2 ) cm x 45 cm

= [tex] \frac {1} {1}[/tex] x 9 cm x 45 cm

= 405 cm²

Luas sisi tegak limas ABT = luas sisi tegak limas DCT, yaitu :

= [tex] \frac {1} {2}[/tex] x alas x tinggi  

= [tex] \frac {1} {2}[/tex] x 32 cm x 43 cm

= [tex] \frac {1} {2 :2}[/tex] x ( 32 : 2 ) cm x 43 cm

= [tex] \frac {1} {1}[/tex] x 16 cm x 43 cm

= 688 cm²

Luas permukaan limas adalah :

= jumlah luas sisi tegak limas + luas alas  

= ( 2 x 405 cm² ) + ( 2 x 688 cm² ) + 576 cm²

= 810 cm² + 1.376 cm² + 576 cm²

= 2.762 cm²

c. Volume limas adalah :

= [tex] \frac {1} {3}[/tex] x luas alas x tinggi

= [tex] \frac {1} {3}[/tex] x ( panjang x lebar ) x tinggi

= [tex] \frac {1} {3}[/tex] x ( 32 cm x 18 cm ) x 42 cm

= [tex] \frac {1} {3 : 3}[/tex] x ( 32 cm x 18 cm ) x ( 42 : 3 ) cm

= [tex] \frac {1} {1}[/tex] x 576 cm² x 42 cm

= 24.192 cm

13. keliling alas 28 cm tingginya 10,5 lplberapakah luas alas tersebut?​


Jawaban:

147 cm²

Penjelasan dengan langkah-langkah:

Untuk menghitung luas alas suatu bangun ruang, kita perlu mengetahui bentuk bangun ruang tersebut. Namun, karena informasi yang diberikan hanya mengenai ukuran alas, maka kita asumsikan bangun ruang tersebut adalah segitiga.

Luas segitiga dapat dihitung dengan rumus:

Luas = 1/2 x alas x tinggi

Diketahui bahwa alas segitiga adalah 28 cm dan tingginya adalah 10,5 cm. Maka, luas alas tersebut dapat dihitung sebagai berikut:

Luas = 1/2 x 28 cm x 10,5 cm

Luas = 14 cm x 10,5 cm

Luas = 147 cm²

Jadi, luas alas tersebut adalah 147 cm².
