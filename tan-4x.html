31. Nilai dari lim X-0 tan 9x + tan 3x 4x adalah .....​


Jawaban:

Untuk menghitung nilai dari limit lim(x→0) [tan(9x) + tan(3x)] / 4x, kita dapat menggunakan aturan L'Hôpital karena kita memiliki bentuk "0/0" pada limit ini.

Langkah-langkahnya adalah sebagai berikut:

1. Pertama, kita coba substitusi x = 0 ke dalam persamaan. Ini memberikan bentuk "0/0", yang tidak dapat diselesaikan secara langsung.

2. Kami akan mengambil turunan dari pembilang dan penyebut terlebih dahulu.

a. Turunan dari tan(9x) adalah 9sec^2(9x).

b. Turunan dari tan(3x) adalah 3sec^2(3x).

c. Turunan dari 4x adalah 4.

3. Sekarang kita dapat menyusun ulang limit dengan turunan yang kita hitung:

lim(x→0) [9sec^2(9x) + 3sec^2(3x)] / 4

4. Sekarang kita substitusi x = 0 ke dalam limit baru ini:

[9sec^2(0) + 3sec^2(0)] / 4

Sekarang, sec^2(0) adalah 1, sehingga persamaan ini menjadi:

[9(1) + 3(1)] / 4

9 + 3 = 12

Jadi, nilai dari limit lim(x→0) [tan(9x) + tan(3x)] / 4x adalah 12.

39. sec^4x–tan^4x=sec^2x+tan^2x buktikan identitas trigonometri


sec^4x - tan^4x = 1/cos^4x - sin^4x/cos^4x

= (1 - sin^4x)/cos^4x
= (cos^2x + sin^2x - sin^2x.sin^2x)/(cos^2x.cos^2x)
= (cos^2x + sin^2x(1 - sin^2x))/(cos^2x.cos^2x)
= (cos^2x + sin^2x.cos^2x)/(cos^2x.cos^2x)
= cos^2x(1 + sin^2x)/(cos^2x.cos^2x)
= (1 + sin^2x)/(cos^2x)
= sec^2x + tan^2x
Q.E.D.
