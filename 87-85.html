4. 25. Rata-rata nilai ulangan IPS 18 siswa adalah 85,5. Rata-rata nilai ulangan IPS menjadi 86 setelah nilai Dayu, Edo, dan Udin dimasukkan. Nilai Dayu 8 lebih dari nilai Edo. Nilai Udin 2 kurang dari nilai Edo. Nilai Dayu, Edo, dan Udin berturut-turut adalah ... A. 85, 87, dan 95 B. 85, 95, dan 87 C. 87, 95, dan 85 D. 95, 87, dan 85boleh minta tlg kerjain ga kak?:)​


Jawab:
(d.)  95, 87, 85

Penjelasan dengan langkah-langkah:
Diketahui data sebelumnya:
Rata-rata (m) = 85,5
Jumlah siswa (f) = 18

Cari jumlah nilai (n)
n = m×f
n = 85,5×18
n = 1.539

Nilai rata-rata baru (mb) = 86
fb = f + 3 = 18+3 = 21
Jumlah siswa baru (fb) = 21

D = Nilai Dayu
E = Nilai Edo
U = Nilai Udin

[Jumlah nilai baru]
n       + D + E + U   = mb × fb
1.539 + D + E + U  = 86×21
1.539 + D + E + U  = 1.806
maka, . . .
D + E + U = 1.806 - 1.539
D + E + U = 267

D = E + 8, maka E = D - 8
U = E - 2, maka U = (D-8)-2, U = D-10

Masukkin ke yg barusan
D + E + U = 267
D + (D-8) + (D-10) = 267
3D - 18 = 267
3D = 267 + 18
3D = 285
D = 285/3
D = 95
Nilai Dayu = 95

Cari nilai Edo (E)
E = D - 8, maka
E = 95 - 8
E = 87
Nilai Edo = 87

Cari nilai Udin (U)
U = E - 2
U = 87 - 2
U = 85
Nilai Udin = 85

Nilai Dayu, Edo, Udin
= 95, 87, 85 (opsi d.)

16. 25. Rata-rata nilai ulangan IPS 18 siswa adalah 85,5. Rata-rata nilai ulangan IPS menjadi 86 setelah nilai Dayu, Edo, dan Udin dimasukkan. Nilai Dayu 8 lebih dari nilai Edo. Nilai Udin 2 kurang dari nilai Edo. Nilai Dayu, Edo, dan Udin berturut-turut adalah ... A. 85, 87, dan 95 B. 85, 95, dan 87 C. 87, 95, dan 85 D. 95, 87, dan 85 N​


Jawab:

95, 87, dan 85 (d.)

Penjelasan dengan langkah-langkah:

Diketahui:

Rata-rata 18 siswa = 85,5Rata-rata 21 siswa = 86Nilai Dayu = NIlai Edo + 8Nilai Udin = Nilai Edo - 2

Ditanya:

Nilai Dayu, Edo, dan Udin

Jawab:

Menentukan jumlah nilai 18 siswa.

Rata-Rata = Jumlah Data / Frekuensi

85,5 = Jumlah Data / 18

Jumlah Data = 85,5 x 18

Jumlah Data = 1539

Jadi jumlah nilai 18 siswa adalah 1539.

Menentukan jumlah nilai Dayu, Edo, dan Udin

Anggap jumlah nilai Dayu, Edo, dan Udin sebagai z.

Rata-Rata = Jumlah Data / Frekuensi

86 = (1539 + z) / (18 + 3)

86 = (1539 + z) / 21

1539 + z = 86 x 21

z = 1806 - 1539

z = 267

Jadi jumlah nilai Dayu, Edo, dan Udin adalah 267.

Menentukan nilai Dayu, Edo, dan Udin

Nilai Dayu = NIlai Edo + 8

Persamaan 1 :  d = e + 8

Nilai Udin = Nilai Edo - 2

Persamaan 2 :  u = e - 2

Jumlah nilai Dayu, Edo, dan Udin = 267

Persamaan 3 :  d + e + u = 267

Subtitusikan persamaan 1 dan 2 ke persamaan 3.

d + e + u = 267

(e + 8) + e + (e - 2) = 267

3e = 267 - 6

e = 261/3

e = 87

Maka nilai Edo adalah 87.

Subtitusikan nilai e ke persamaan 1.

d = e + 8

d = 87 + 8

d = 95

Maka nilai Dayu adalah 95.

Subtitusikan nilai e ke persamaan 2.

u = e - 2

u = 87 - 2

u = 85

Maka nilai Udin adalah 85.

Jadi, nilai Dayu, Edo, dan Udin berturut-turut adalah 95, 87, 85.

19. Rumus suku ke-n dar barisan 89, 87, 85, 83,adalah....​


Penjelasan dengan langkah-langkah:

a = 89

b = 87 - 89 = -2

Un = a + (n - 1)·b

Un = 89 + (n - 1)·(-2)

Un = 89 - 2n + 2

Un = 91 - 2n

⏩ Penyelesaian Soal:

- Diketahui:

Barisan Aritmatika: 89, 87, 85, 83,..... a = 89 b = U2 - U1 = 87 - 89 = -2

- Ditanya:

Rumus Suku ke-n

- Jawab:

~ Rumus: a + ( n - 1 ) b

= a + ( n - 1 ) b

= 89 + ( n - 1 ) -2

= 89 + ( -2n ) + 2

= 89 - 2n + 2

= 89 + 2 - 2n

= 91 + n

Kesimpulan:

- Maka Rumus suku ke-n dari Barisan Aritmatika adalah 91 + n

Pembahasan :

Kaidah pencacahan merupakan suatu cara aturan yang digunakan untuk menghitung banyaknya hasil yang memungkinkan dari suatu percobaan.

Aturan pencacahan terdiri dari 4 jenis bagian yaitu :

✔️ Aturan perkalian ( Filling Slots )

Prinsipnya mengalikan banyak kejadian yang mungkin dari tiap penyusunnya.

✔️ Faktorial.

Faktorial adalah sebuah fungsi permutasi dari bilangan bulat positif yang menggunakan notasi n! . Faktorial akan mengalikan bilangan dengan bilangan berikutnya hingga terakhir dikalikan angka 1. Faktorial mengikuti rumus berkikut.

✔️ Permutasi.

Permutasi adalah penyusunan kembali suatu kumpulan objek dalam urutan yang berbeda dari urutan yang semula.

Rumus Permutasi Yaitu :

➡️ n! ÷ k!

➡️ n! / k!

Keterangan :

▶️ n! = Jumlah Huruf

▶️ k! = unsur berulang / unsur ganda

✔️ Kombinasi.

Kombinasi adalah menggabungkan beberapa objek dari suatu grup tanpa memperhatikan urutan. Di dalam kombinasi, urutan tidak diperhatikan. {1,2,3} adalah sama dengan {2,3,1} dan {3,1,2}.

Rumus Kombinasi yaitu :

C = n! ÷ r! ( n - r )!

Keterangan :

▶️ n! = jumlah huruf

▶️ r! = jumlah objek yang dipilih dari kumpulan

38. Tiga bilangan ganjil berurutan yang jumlahnya sama dengan 255 adalah ... *A.81, 85, 89B.81, 83, 85C.80, 85, 90D.83, 85, 87​


Jawaban:

D. 83, 85, 87

Penjelasan:

Tiga bilangan ganjil berurutan yang jumlah nya 255

x + x + 2 + x + 4 = 255

3x + 6 = 255

3x = 255 - 6

3x = 249

x = 249 : 3

x = 83

Bilangan 1

x = 83

Bilangan 2

x + 2

= 83 + 2

= 85

Bilangan 3

x + 4

= 83 + 4

= 87

Jadi, 3 bilangan ganjil berurutan yang jumlahnya sama dengan 255 adalah D. 83,85,87

Jawabannya adalah D. 83,85,87

Pembahasan

Tiga Bilangan Ganjil berurutan yang jumlahnya sama dengan 255

Bilangan aritmatika dengan cara

- Bilangan 1 = x

- Bilangan 2 = x + 2

- Bilangan 3 = x + 4

Jadi dalam satu rangkaian bilangan yaitu

x + x + 2 + x + 4 = 255

3x + 6 = 255

3x = 255 - 6

3x = 249

x = 83

Jadi Bilangan Pertama = 83

Bilangan Kedua = 83 + 2 = 85

Bilangan Ketiga = 83 + 4 = 87
