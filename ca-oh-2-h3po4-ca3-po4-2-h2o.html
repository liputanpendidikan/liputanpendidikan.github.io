Pembahasan

[tex]\boxed{\boxed{\bold{Persamaan~Reaksi }}}[/tex]

Persamaan reaksi adalah proses kimia yang menjelaskan terjadinya reaksi antara zat reaktan dan zat produk. Antara zat reaktan dan zat produk dipisahkan dengan tanda panah (--->) yang menunjukkan arah reaksi. Kemudian persamaan reaksi juga menuliskan wujud zat yaitu (g) berarti gas, (l) berarti liquid/cairan, (aq) berarti larutan (aqueous), (s) berarti padatan (solid). Persamaan reaksi setara ketika persamaan reaksi sudah diberikan koefisien reaksi. Koefisien reaksi adalah bilangan angka yang terletak mendahului rumus kimia zat.  

[tex]\boxed{\boxed{\bold{Diket}}}[/tex]  

Persamaan reaksi yang terjadi adalah

Ca(OH)₂ + H₃PO₄ --> Ca₃(PO₄)₂ + H₂O

[tex]\boxed{\boxed{\bold{Ditanya}}}[/tex]  

Koefisien reaksi masing - masing

1. Menentukan koefisien salah satu zat, biasanya zat yang memiliki rumus paling kompleks ditetapkan memiliki koefisien 1, sedangkan zat yang lain diberikan koefisien dengan huruf

a Ca(OH)₂ + b H₃PO₄ --> Ca₃(PO₄)₂ + c H₂O

kita misalkan Ca₃(PO₄)₂ sebagai zat yang paling kompleks dan memiliki koefisien reaksi 1 sedangkan zat yang lainnya kita ganti dengan huruf a, b dan c.

2. Menyetarakan terlebih dahulu unsur yang terikat langsung dengan zat yang diberikan koefisien 1

Menyetarakan atom Ca

jumlah atom kiri = jumlah atom kanan

a = 3

3. Menyetarakan unsur  lain yang masih belum setara, mengusahakan menyetarakan atom O dan H ditahap yang paling terakhir.

a Ca(OH)₂ + b H₃PO₄ --> Ca₃(PO₄)₂ + c H₂O

Menyetarakan atom H

jumlah atom kiri = jumlah atom kanan

2a + 3b = 2c

2. 3 + 3. 2 = 2c

6 + 6 = 2c

Menyetarakan atom O

jumlah atom kiri = jumlah atom kanan

2a + 4b = 8 + c

2. 3 + 4. 2 = 8 + c

6 + 8 = 8 + c

c = 6

20. Ca (oh)2 + H3po4 maka ca3(po4)2+H2o Setarakan pers reaksi


Jawaban:

3 Ca(OH)2 + 2 H3PO4 => Ca3(PO4)2 + 6 H2O

Penjelasan:

Gunakan masing" abjad untuk tiap koefisien:

a Ca(OH)2 + b H3PO4 => c Ca3(PO4)2 + d H2O

a Ca = c Ca3

maka a = 3

b PO4 = c (PO4)2

maka b = 2

Menyetarakan jumlah atom H

3 Ca(OH)2 => 6 atom H

2 H3PO4 => 6 atom H

Total atom H ruas kiri = 12

12 atom H = d H2O

maka d = 6

Menyetarakan jumlah atom O

Ruas kiri:

3 Ca(OH)2 => 6 atom O

2 H3PO4 => 8 atom O

Total atom O ruas kiri = 14

Ruas kanan

6 H2O => 6 atom O

Misal c = 1 maka Ca3(PO4)2 => 8 atom O

Total atom O ruas kanan = 14 ( sudah setara )

Jadi persamaan reaksinya adalah:

3 Ca(OH)2 + 2 H3PO4 => Ca3(PO4)2 + 6 H2O
