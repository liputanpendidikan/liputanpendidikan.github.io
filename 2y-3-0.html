37. Bayangan garis x + 2y – 1 = 0 jika ditranslasioleh T = (2,-3)adalah ..A. X + 2y + 2 = 0 D. 2x + y + 3 = 0B. X + 2y + 3 = 0 E. 3x + 2y + 3 = 0C. 2x + y + 2 = 0​

Penjelasan dengan langkah-langkah:

Translasi atau pergeseran, rumus yang dipakai untuk menentukan suatu bayangan yang telah ditranslasi adalah sebagai berikut.

P(x,y) ---> translasi T(a, b) ---> P(x', y') = (x + a, y + b)

Berdasakan soal tersebut,

P(x,y) ---> translasi T(2, -3) ---> P(x', y') = (x + 2, y - 3)

diperoleh nilai x' = x + 2 ---> x = x' - 2

                       y' = y - 3 ---> y = y' + 3

substitusikan nilai x dan y ke persamaan garis x + 2y - 1 = 0

x' - 2 + 2(y' + 3) - 1 = 0

x' - 2 + 2y' + 6 - 1 = 0

x' + 2y' + 3 = 0

Jadi, hasil translasi garis m adalah x + 2y + 3 = 0. (B)
