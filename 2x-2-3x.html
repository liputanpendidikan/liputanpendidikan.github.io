9. 22. Fungsi komposisi (gof) (x) = 8x² + 2x + 1. Jika f(x) = 2x + 1 maka g(x) = .... A. 2x² + 3x + 2 D. 2x² 3x + 2 B. 2x²-3x - 2 E. 3x² + 2x - 2 C. 3x² - 2x + 2 -​


Penjelasan dengan langkah-langkah:

Untuk mencari g(x) dari fungsi komposisi gof(x), kita perlu menggantikan setiap x dalam fungsi f(x) dengan fungsi f(x) itu sendiri (yaitu 2x + 1).

Sehingga,

g(x) = 8(2x + 1)² + 2(2x + 1) + 1

Mari kita selesaikan langkah-langkah perhitungannya:

1. Kuadratkan (2x + 1):

(2x + 1)² = (2x + 1)(2x + 1) = 4x² + 2x + 2x + 1 = 4x² + 4x + 1

2. Kalikan dengan 8:

8(4x² + 4x + 1) = 32x² + 32x + 8

3. Kalikan dengan 2:

2(2x + 1) = 4x + 2

Setelah itu, gabungkan semua hasilnya:

g(x) = 32x² + 32x + 8 + 4x + 2 + 1

g(x) = 32x² + 36x + 11

Jadi, g(x) = 32x² + 36x + 11

Opsi yang sesuai adalah E. 32x² + 36x + 11.
