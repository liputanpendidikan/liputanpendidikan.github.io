Pembahasan soal no. 2

Sebuah kereta api pertama melaju dari kota A ke kota B selama 4 jam dengan kecepatan rata-rata 72km/jam. Jika kereta kedua melaju pada lintasan yang sama dalam waktu 6 jam, berapa kecepatan rata-rata kereta api kedua tersebut?

Cara pertama:

Jarak = kecepatan x waktu

= 72 km/jam x 4 jam

= 288 km

Jika kereta kedua melaju pada lintasan yang sama, berarti jarak yang ditempuh kereta api kedua sama dengan jarak yang ditempuh kereta api pertama

Kecepatan = jarak/waktu

= 288 km/6 jam

= 48 km/jam

Cara kedua:

4 jam → 72 km/jam

6 jam → x

x1/y1 = y2/x2

2/3 = x/72

3x = 144

x = 48 km/jam
