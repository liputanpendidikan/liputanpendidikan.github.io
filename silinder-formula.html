100 / 4² = 400 / d2²
1 / 4² = 4 / d2²
d2² = 4² × 4 = 64
d2 = √64 = 8 cm P₁ = P₂
F₁ / A₁ = F₂ / A₂
F₁ / (1/4 πD₁²) = F₂ / (1/4 πD₂²)
D₂ = D₁ √(F₂ / F₁)
     = (4 cm) √[400 N / (100 N)] = 8 cm

Jawaban:

Diameter silinder besar adalah 18√2 cm

Penjelasan:

F₁ = 700 N

F₂ = 5600 N

D₁ = 9 cm → r₁ = 9/2 cm

Ditanya :

Diameter silinder besar = ___?

Jawab :

Konsep Tekanan dimana kedua silinder mengalami tekanan yang sama, sehingga jari jari silinder besar :

P₁ = P₂

F₁/πr₁² = F₂/πr₂²

700/r₁² = 5600/r₂²

700/(9/2)² = 5600/r₂²

2800/81 = 5600/r₂²

1/81 = 2/r₂²

r₂² = 162

r₂ = 9√2 cm

Diameter silinder besar :

D₂ = 2 × 9√2

D₂ = 18√2 cm


8. Contoh Mengubah rumus formula matematika lama menjadi formula baru


Hmm... ada sebenarnya satu cara (bukan rumus baru ataupun lama sih) yang menarik yang bisa dicari dengan cara dari bab lain
biasanya untuk mencari titik puncak x dari fungsi kuadrat kan kayak gini :
[tex]Xp = \frac{x_{1}+x_{2}}{2} [/tex] (ini digunakan kalau ada nilai titik potong x) atau [tex]Xp = - \frac{b}{2a} [/tex] (dan ini digunakan untuk persamaan ax²+bx+c)
tapi mencari titik puncak x juga bisa dicari dengan konsep dari bab turunan (Kelas 11) : Misalkan f(x) = 2x² - 3x + 4
Xp = f'(x)
f'(x) = 4x - 3
4x   = 3
  x   = [tex] \frac{3}{4} [/tex]
Pembuktian dengan rumus asli Xp :
Xp = [tex]- \frac{b}{a} [/tex]
      = [tex]- \frac{3}{2.2} [/tex]
      = [tex] \frac{3}{4} [/tex] (Terbukti)

Jawaban:

Penjelasan:

SIMPLE PRESENT TENSE

merupakan tenses yang digunakan mendeskripsikan kejadian nyata di masa lampau, saat ini dan akan tetap selalu sama  di tahun - tahun mendatang.

Tenses ini juga menjabarkan fakta kebenaran umum, kebenaran yang tak terbatas dan aktivitas yang terus- menerus berulang  ( kebiasaan/habit/ hobby )

Penggunaan keterangan frekuensi ( Adverb of frequency) yang menjadi ciri khas utama dalam menyatakan habit/kebiasaan dalam simple present tense  :

Always, usually, often, everyday, seldom, rarely, daily, never, frequently, occasionally , ever, sometimes, regularly.

PATTERN

Struktur kalimat dikategorikan ke dalam 2 jenis, Verbal dan nominal.

1. Rumus Simple present tense ( verbal sentence )

Verbal sentence merupakan kalimat yang predikatnya berupa kata kerja aksi/tindakan.

(+) SUBJECT + VERB (S/ES ) + OBJECT/COMPLEMENT

(-)  SUBJECT + DO/DOES +NOT + VERB 1 + OBJECT /COMPLEMENT

(?) DO/DOES+SUBJECT + VERB 1 + OBJECT / COMPLEMENT?

Example :

(+ ) Dino often reads some books in the library   ( Habit/kebiasaan/ hobby)

( - ) Dino  doesn't often read some books in the library .

(? ) Does Dino often  read some books in the library?

2. Rumus simple present tense ( nominal sentence )

Nominal sentence merupakan kalimat yang predikatnya bukan berupa kata kerja aksi. Pada umumnya berupa kata benda ( noun), kata sifat ( adjective ), dan keterangan ( adverb )

(+ ) SUBJECT + TOBE (IS/AM/ARE)+NOUN/ADJECTIVE/ADVERB

(- ) SUBJECT + TOBE(IS/AM/ARE ) + NOT + NOUN/ADJECTIVE/ADVERB

( ? ) TOBE(IS/AM/ARE ) + SUBJECT+NOUN/ADJECTIVE/ADVERB?

Example :

( + ) Dino is smart.  ( penjabaran karakter, bagian sifat )

( - ) Dino is not smart.

( ? ) Is Dino smart ?

Penggunaan Urea :
0,6 x + 0,7 y = 190

Formula II dapat dibuat sebanyak 100 kg atau 1,0 kwintal

0,4 x + 0,3 y = 110
0,4 x + 0,3 (100) = 110
0,4 x + 30 = 110
0,4 x = 110 - 30
0,4 x = 80
x = (80) / (0,4)
x = 200

Jawaban:

4576 cm²

Penjelasan dengan langkah-langkah:

diketahui :

t = 38 cm

r = 14 cm

luas silinder = 2.luas alas + luas selimut

kita cari dulu luas alas (atas dan bawah)

luas alas = luas lingkaran

luas alas = πr²

luas alas = (22/7)14²

luas alas = 616 cm²

kita cari luas selimutnya

luas selimut = keliling lingkaran x tinggi

luas selimut = 2πr x t

luas selimut = 2(22/7)14 x 38

luas selimut = 88 x 38

luas selimut = 3.344 cm²

luas silinder = 2.luas alas + luas selimut

luas silinder = 2.616 cm² + 3.344 cm²

luas silinder = 1.232 cm² + 3.344 cm²

luas silinder = 4576 cm²

35. yang berfungsi sebagai penutup lubang silinder pada lubang silinder pada Blok silinder dan tempat dudukan Busi yaitu.. A:katupB:Blok silinderC:ChamsaftD:pistonE:kepala silinderjawab sekarang yah plis​


Sebagai salah satu komponen utama mesin motor, kepala silinder berfungsi sebagai tempat dudukan busi dan untuk menutup lubang silinder yang ada pada blok silinder. Komponen ini dipasang dan disekat pada titik tumpu oleh gasket agar kompresi tidak bocor. Jawaban berdasarkan soal adalah (E).

Pembahasan

Ada beberapa bagian komponen pada kendaraan, khususnya sepeda motor. Bagian utama komponen tersebut adalah cylinder header (kepala silinder) dan cylinder block (blok silinder).

Bagi mesin sepeda motor, kedua komponen tersebut merupakan tulang punggung atau alat paling penting dalam menggerakkan mesin motor.  Kedua komponen utama tersebut digunakan pada mesin motor dua langkah (dua tak) dan empat langkah (empat tak). Kepala silinder salah satu jenis komponen utama yang ada pada mesin motor, yang diikat dengan menggunakan baut dan dipasangkan pada blok silinder. Adapun komponen kepala silinder dan fungsinya adalah sebagai berikut:

Tutup oli
Komponen ini merupakan komponen paling atas yang ada pada kepala silinder. Fungsi dari tutup oli adalah sebagai penutup sehingga oli tidak keluar ketika kendaraan sedang beroprasi. Selain itu, tutup oli juga berfungsi sebagai tempat memasukkan oli saat oli mesin diganti.
Tutup Kepala Silinder (Cylinder Head Cover)
Tutup Kepala Silinder berfungsi untuk menutup kepala silinder sehingga dapat mencegah oli mesin tumpah ketika mesin sedang beroperasi. Selain itu, komponen ini juga berfungsi untuk melindungi komponen mekanisme katup pada kepala silinder dari berbagai macam kotoran. Tutup kepala silinder juga difungsikan sebagai tempat untuk dudukan kabel busi.
Busi (Spark Plug)
Busi merupakan komponen kepala silinder yang berfungsi untuk menghasilkan loncatan bunga api listrik. Hal tersebut dibutuhkan untuk membakar campuran bahan bakar dan udara. Jangan lupa untuk selalu mengecek kondisi busi motor Anda.
Penyetel Celah Katup (Adjusting Shim Valve)
Penyetel Celah Katup berfungsi untuk mengatur celah katup. Alat untuk menyetel celah katup terdiri dari obeng min (-), Fuller Gauge, dan kunci ring 10.
Penghantar Katup (Valve Guide)
Komponen kepala silinder dan fungsinya yang berikutnya adalah penghantar katup. Penghantar Katup ini berfungsi untuk menghantarkan gerakan katup.
Jaket Air (Water Jacket)
Jaket air bertungsi sebagai saluran air pendingin agar komponen mesin tetap dingin saat sedang beroperasi.
Pengangkat Katup (Valve Lifter)
Pengangkat katup berfungsi untuk mengangkat katup.
Katup Buang (Exhauts Valve)
Katup buang memiliki fungsi sebagai tempat keluarnya gas buang dari ruang pembakaran.
Katup Masuk (Intake Valve)
Katup masuk ini berfungsi sebagai tempat masuknya bahan bakar yang bercampur dengan udara di dalam ruang bakar.
Dudukan Katup (Valve Seat)
Dudukan katup berfungsi sebagai dudukan dari kepala katup.
Sil Oli (Oil Seal)
Sil oil berfungsi sebagai perapat oli sehingga oli tidak akan masuk ke ruang pembakaran.
Pengunci Katup (Valve Keepers)
Komponen kepala silinder dan fungsinya selanjutnya adalah pengunci katup. Pengunci katup ini berfungsi untuk mengunci antara pegas katup dengan katupnya.
Pegas Katup (Valve Spring)
Pegas katup berfungsi untuk mengembalikan katup pada posisi semula.
Rocker Arm Shaft
Komponen ini berfungsi sebagai tempat dudukan rocker arm.
Rocker Arm
Rocker arm befungsi sebagai penggerak atau penekan katup masuk atau katup buang.
Gasket
Komponen yang satu ini berfungsi untuk merapatkan kepala silinder dengan blok silinder sehingga air pendingin, kompresi, maupun oli tidak akan mengalami kebocoran.
Ruang Bakar (Combustion Chamber)
Komponen ini berfungsi sebagai tempat terjadinya proses pembakaran bahan bakar.
