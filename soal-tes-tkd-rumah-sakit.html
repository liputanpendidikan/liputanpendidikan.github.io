Indikator : menentukan fungsi limit jika diketahui nilai limit aljabar bentuk tak tentu

lim x→a (x² - bx)/(a - x) = 2

Bentuk tak tentu:
untuk x = a maka (a² - ab) = 0
a(a - b) = 0
a = 0 atau a = b

Untuk bentuk l'hospital:
(2x - b)/(-1) = 2
2a - b = -2
jika a = b, maka
a = -2,
b = -2

maka ab = 4
