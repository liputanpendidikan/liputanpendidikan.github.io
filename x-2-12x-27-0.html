9. 27. Bentuk bentuk berikut merupakan kuadrat sempurna, kecuali .... a. x ^ 2 - 8x + 16 = 0 b. x ^ 2 + 14x + 49 = 0 c. x ^ 2 - 10x + 20 = 0 d. x ^ 2 + 12x + 36 = 0


Jawaban:

persamaan yang bukan merupakan kuadrat sempurna adalah pilihan (c) x^2 - 10x + 20 = 0.

Penjelasan dengan langkah-langkahnya:

Berikut adalah langkah-langkah untuk menentukan apakah persamaan kuadrat merupakan kuadrat sempurna atau tidak:

1. **Pahami Bentuk Kuadrat Sempurna**: Kuadrat sempurna adalah bentuk persamaan kuadrat yang dapat ditulis dalam bentuk (x - a)^2 = 0, di mana "a" adalah suatu bilangan.

2. **Periksa Setiap Pilihan**: Periksa setiap pilihan persamaan kuadrat yang diberikan:

a. x^2 - 8x + 16 = 0

b. x^2 + 14x + 49 = 0

c. x^2 - 10x + 20 = 0

d. x^2 + 12x + 36 = 0

3. **Cobalah untuk Faktorkan**: Cobalah untuk faktorkan setiap persamaan kuadrat tersebut menjadi bentuk (x - a)^2 = 0. Jika Anda dapat melakukan ini, maka itu adalah kuadrat sempurna.

a. x^2 - 8x + 16 = (x - 4)^2 = 0

b. x^2 + 14x + 49 = (x + 7)^2 = 0

c. x^2 - 10x + 20 = Tidak dapat difaktorkan menjadi (x - a)^2

d. x^2 + 12x + 36 = (x + 6)^2 = 0

4. **Kesimpulan**: Berdasarkan faktorisasi di atas, pilihan (c) x^2 - 10x + 20 = 0 tidak dapat difaktorkan menjadi bentuk (x - a)^2 = 0, sehingga persamaan ini bukan kuadrat sempurna.

Jadi, persamaan yang bukan merupakan kuadrat sempurna adalah pilihan (c) x^2 - 10x + 20 = 0.

14. X² - 12x + 27 = 0 Tentukan rumus ABC dan pemfaktoran​


Persamaan Kuadrat Dari x² - 12x + 27 = 0 adalah HP { 9,3 }

Pembahasan :

Persamaan Kuadrat merupakan suatu persamaan dari variabel yang mempunyai pangkat tertinggi dua.

Bentuk Umum :

ax² + bx + c = 0

a ≠ 0

a,b,c anggota bilangan Rill

Contoh :

x² + 5x + 6 = 0

Variabel : x² dan x

Koefisien : 1 dan 5

Konstanta : 6

1) Menentukan Akar persamaan kuadrat dengan pemfaktoran

ax² + bx + c = 0, dengan a = 1

Contoh :

x² + 8x + 12 = 0

( x + 6 ) ( x + 2 ) = 0

x + 6 = 0 dan x + 2 = 0

x1 = 6 dan x2 = 2

Maka X1 = -6 Dan X2 = -2

Note :

P + Q = b (P + Q = 8)

P × Q = c (P + Q = 12)

2) Menentukan Akar Persamaan Kuadrat Dengan Rumus ABC

ax² + bx + c = 0

[tex] x1.2 = \frac{ - b \:± \sqrt{ {b}^{2} - 4ac } }{2a} [/tex]

3) Menentukan Persamaan Kuadrat Jika Diketahui Akar Akarnya

a) Persamaan Kuadrat Jika akar X1 dan X2

( x - X1 ) ( x - X2) Atau x² - ( X1 + X2 )x + (X1 . X2)

b) Persamaan Kuadrat ax² + bx + c = 0 dapat berubah menjadi :

[tex] {x}^{2} + \frac{ b}{a} x + \frac{c}{a} = 0[/tex]

dalam keterangan :

[tex]x1 + x2 = - \frac{b}{a} \\ \\ x1.x2 = \frac{c}{a} [/tex]

3) Diskriminan

D = b² - (4ac) untuk ax² + bx + c = 0

Contoh : x² + 3x - 10 = 0

a = 1

b = 3

c = -10

D = 3² - (4.1.(-10))

D = 9 - (-40)

D = 9 + 40

D = 49

Jadi D > 0

Penyelesaian:

Pemfaktoran

x² - 12x + 27 = 0

( x - 9 ) (x -3) = 0

x - 9 = 0 dan x - 3 = 0

x1 = 9 dan x2 = 3

HP {9,3}

Rumus ABC

a = 1

b = -12

c = 27

[tex]x1.2 = \frac{ - b± \sqrt{ {b} ^{2} - 4ac } }{2a} \\ \\ x1.2 = \frac{ - ( - 12)±\sqrt{ { - 12}^{2} - 4.1.27} }{2.1} \\ \\x 1.2 = \frac{ 12±\sqrt{144 - 108} }{2} \\ \\ x1.2 = \frac{12± \sqrt{36} }{2} \\ \\ x1.2 = \frac{12 ±6 }{2} \\ \\ x1 = \frac{12 + 6}{2} = \frac{18}{2} = 9 \\ \\ x2 = \frac{12 - 6}{2} = \frac{6}{2} = 3[/tex]

Jadi Persamaan Kuadrat Dari x² - 12x + 27 = 0 adalah HP { 9,3 }

29. carilah himpunan dari 12x^2+4x = 016x^2-12x = 0x^2-5x = 0​


Jawab:

Mari kita selesaikan persamaan-persamaan tersebut satu per satu:

1. [tex]\(12x^2 + 4x = 0\)[/tex]

Faktorkan [tex]\(4x\)[/tex] dari kedua suku:

[tex]\(4x(3x + 1) = 0\)[/tex]

Dari sini kita memiliki dua solusi:

a) [tex]\(4x = 0 \rightarrow x = 0\)[/tex]

b) [tex]\(3x + 1 = 0 \rightarrow 3x = -1 \rightarrow x = -1/3\)[/tex]

Jadi, himpunan solusi pertama adalah [tex]\({0, -1/3}\)[/tex].

2. [tex]\(16x^2 - 12x = 0\)[/tex]

Faktorkan [tex]\(4x\)[/tex] dari kedua suku:

[tex]\(4x(4x - 3) = 0\)[/tex]

Dari sini kita memiliki dua solusi:

a) [tex]\(4x = 0 \rightarrow x = 0\)[/tex]

b) [tex]\(4x - 3 = 0 \rightarrow 4x = 3 \rightarrow x = 3/4\)[/tex]

Jadi, himpunan solusi kedua adalah [tex]\({0, 3/4}\)[/tex].
